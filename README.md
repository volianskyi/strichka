# strichka
## Spring MVC project
Simple mvc project about movies data. You can do crud operation on such entit—àes movie, actor, director, user. 
Add, update, delete relations between movies and actors, genres, directors. Sort movies by country.

## Used technologies
* Java 11
* Hibernate
* MySql
* Spring Web MVC
* Spring ORM
* Spring Security
* Spring Exception Handling
* Thymeleaf
* Bootstrap
* Log4j

Project has two user with different roles:
| Login         | Password | Role  |
| ------------- |:------:|:-----:|
| admin | 1111   | ADMIN |
| user | 2222   | USER  |


Admin has access to all resources, permission to change entities.
User can only read data and update his account by himself.
